# Retail Sales Performance Analysis Dashboard

## Project Overview

This project analyzes retail transaction data to understand business performance, customer purchasing behavior, and product profitability.
An interactive Excel dashboard was created to help stakeholders quickly monitor sales, profit, orders, and overall business health.

The goal of the project is to transform raw transactional data into meaningful business insights using data cleaning, analysis, and visualization techniques.

---

## Dataset

The dataset contains retail store order records including:

* Order ID
* Order Date
* Ship Date
* Region
* Product Category
* Sub Category
* Sales Amount
* Quantity
* Discount
* Profit

The data was processed and structured for analysis and reporting.

---

## Tools & Technologies Used

* Microsoft Excel (Data Cleaning, Pivot Tables, Dashboard)
* SQL (Business Query Analysis)
* Data Visualization Techniques
* GitHub (Project Documentation & Portfolio)

---

## Data Preparation

The following steps were performed before analysis:

* Removed duplicates
* Corrected date formats
* Created calculated columns (Year, Month, Profit %)
* Standardized numerical values
* Prepared data table for pivot analysis

---

## Dashboard Features

The interactive dashboard includes:

* KPI Cards (Total Sales, Total Profit, Total Orders, Profit Margin)
* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Profit by Category
* Region filter (Slicer interaction)

The dashboard allows users to dynamically filter results by region and view business performance instantly.

---

## Key Insights

* Technology category generated the highest revenue
* Some categories produced strong sales but lower profitability
* Business maintained an overall profit margin of ~9%
* Seasonal patterns were observed in monthly sales trends
* Regional performance differences were identified across locations

---

## SQL Analysis

SQL queries were written to simulate real business questions such as:

* Top customers by revenue
* Region-wise sales performance
* Most profitable product categories
* Identification of loss-making products
* Yearly sales trends

All queries are included in the `analysis_queries.sql` file.

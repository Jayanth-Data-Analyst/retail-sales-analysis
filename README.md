# Retail Sales Data Analysis Project

## Objective
Analyze retail business performance and identify sales and profit trends.

## Tools Used
- Excel (data cleaning and pivot tables)
- SQL (data analysis queries)
- Power BI (interactive dashboard)

## Key Insights
- Technology category generated highest profit
- West region had high sales but lower margins
- Few products were causing consistent losses
- Top 20% customers contributed majority revenue

## Dashboard
The Power BI dashboard shows KPIs, regional performance, customer trends and monthly sales patterns.
